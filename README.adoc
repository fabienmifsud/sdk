= Saagie SDK for External Technologies

== How to contribute?

=== Requirements

* Node.js v12.x.x minimum

=== Install dependencies

----
yarn
yarn bs
----

=== Run tests

----
yarn test
----

=== Development

Run the `dev` command at the root of the project

----
yarn dev
----

Then go to your technology folder (here `cd examples/techno-C`) and run:

----
yarn start
----
